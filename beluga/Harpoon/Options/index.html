Module Harpoon.Options

type interaction_mode = [
  1. | `stop
  2. | `go_on
]

The `stop and `go_on flag control what happens in the presence of errors. In particular, the `stop flag will cause Harpoon to exit as soon as an error in encountered instead of continuing to process commands which may not make sense anymore. This is especially important when running tests.

type save_mode = [
  1. | `save_back
  2. | `no_save_back
]

Controls the behaviour of saving sessions back to the signature when they are completed.
