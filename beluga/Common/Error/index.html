type print_result

Abstract dummy datatype to enforce that printing be done using the printing functions provided by this module.

val register_printer : (exn -> print_result) -> unit

Wrapper around Printexc.register_printer. The given function should use partial pattern matching (ew) to signal that it doesn't know how to deal with other types of exceptions.

val register_printer' : (exn -> print_result option) -> unit

Wrapper around Printexc.register_printer. The given function should return None if it doesn't know how to handle a given exception.
